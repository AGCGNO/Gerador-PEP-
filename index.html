Gerador de Estruturas PEP da AGCGNO

PEPs Etapas

1) Dados CAPEX (com cabeçalho)

Cole a planilha com cabeçalho. O sistema identifica automaticamente: Coletor de Custo / Definição / PEP SIGEP, ID real (ou ID SIGEP), Descrição/Denominação, Objeto/Especificação e Local/Usina.

2) Inserir manualmente

Use quando precisar criar um PEP específico sem colar do CAPEX.

Pré-visualização

O novo início redefine apenas o sequencial daquela usina. As demais permanecem iguais.
